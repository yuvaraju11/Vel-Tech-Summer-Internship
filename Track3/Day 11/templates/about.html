📌 Project Overview

The Student Registration System (SRS) is a web application built using Python Flask for the backend and HTML + CSS for the frontend. It allows institutions to register students, view student records, and manage basic academic information — all in one clean interface.

🛠️ Tech Stack

Python 3 Flask HTML5 CSS3 Jinja2 Templates SQLite (Day 2)
