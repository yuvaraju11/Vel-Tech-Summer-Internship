About AQI SmartCity

A real-time and predictive Air Quality Index platform built for the Innolift Ventures Internship — Track 2.

Overview

AQI SmartCity combines two sources of air quality information. The Live AQI page fetches real-time readings from public monitoring stations worldwide. The Forecast page uses a Random Forest model trained on historical Indian AQI data to estimate AQI for a chosen state, area, and month — with results bounded by the actual historical range for that location so forecasts stay realistic.

Features

Real-Time AQI Lookup ML Forecast Engine Dataset Dashboard Forecast History Mobile Responsive CPCB Health Advisories
