# Air Quality Index Predictor - Smart City

## Project Overview
This project predicts air quality categories using machine learning. The model analyzes AQI data and predicts the air quality status based on different environmental factors.

## Dataset
India AQI Dataset

## Features Used
- State
- Area
- Number of Monitoring Stations
- Prominent Pollutants
- AQI Value

## Tasks Completed
- Loaded and analyzed the AQI dataset.
- Performed Exploratory Data Analysis (EDA).
- Cleaned and preprocessed the data.
- Trained a Machine Learning model.
- Evaluated model performance.
- Saved the trained model as model.pkl.
- Predicted AQI categories for sample inputs.
- Created visualizations and charts.

## Algorithm
Random Forest Classifier

## Accuracy
90%+

## How To Run

### Install Required Libraries

pip install pandas numpy matplotlib scikit-learn

### Run Dataset Loading

python load_data.py

### Run EDA

python eda_report.py

### Run Data Cleaning

python clean_data.py

### Train Model

python train_model.py

### Run Prediction

python predict.py
