# Air Quality Index Predictor - Smart City

## Project Overview
This project predicts air quality categories using machine learning.

## Dataset
India AQI Dataset

## Features Used
- State
- Area
- Number of Monitoring Stations
- Pollutants
- AQI Value

## Algorithm
Random Forest Classifier

## Accuracy
90%+

## How To Run

python load_data.py

python eda_report.py

python train_model.py

python predict.py
